# 贡献指南

感谢你愿意为 HelloAgents-Go 贡献代码。

## 贡献原则

- 与 Python 版本行为保持一致
- 保持改动聚焦，避免大而杂的提交
- 不引入无必要的新能力或破坏性改动

## 提交流程

1. Fork 并创建分支
2. 实现改动（代码 + 文档）
3. 运行检查：
```bash
gofmt -w .
go test ./...
go vet ./...
```
4. 提交 PR，说明：
   - 变更目的
   - 对齐依据（对应 Python 模块/方法）
   - 风险与回归点

## PR 要求

- 代码可编译，测试通过
- README / docs 在需要时同步更新
- 不包含无关重构

## Issue 建议模板

- 背景与问题描述
- 复现步骤
- 期望行为
- 当前行为
- 相关日志或截图（如有）
